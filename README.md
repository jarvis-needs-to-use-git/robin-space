# Robin Space | Phased Array Antenna Analysis

A professional-grade tool for analyzing and simulating phased array antennas.

## Features
- **Active Reflection Coefficient (ARC)** Analysis
- **Floquet Unit Cell** Simulation (FDTD)
- **Generalized Scattering Matrix (GSM)** Cascading
- **Beam Steering** Visualization

## Physics Engine
This application uses [Meep](https://meep.readthedocs.io/), a powerful open-source FDTD simulation software, running in a custom Docker container.
